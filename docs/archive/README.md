Folder archiwalny dokumentacji

Ten katalog ma na celu przechowywanie oryginalnych wersji dokumentów oznaczonych jako archiwalne.

Aktualny workflow:

- Nie kasujemy niczego od razu.
- Kopiujemy oryginalne pliki dokumentacji do tego folderu (jeśli wybrano przeniesienie).
- W miejscu oryginału można pozostawić krótką informację: "ARCHIWALNE — patrz docs/archive/<nazwa>".

Jeżeli potwierdzisz, wykonam operację przeniesienia lub wystawienia banerów w plikach.
